Дано натуральное число N, большее 2. Необходимо напечатать прямоугольный равнобедренный треугольник с катетом длиной N, и прямым углом справа снизу.
Объявим переменную n и и запишем ввод пользователя в нее.
Поскольку такой треугольник состоит из n слоев, используем цикл for с i = 0, ограничением i < n и шагом 1.
Для вывода в каждом слое звездочек треугольника используем вложенный цикл for с j = 1, ограничением j <= n и шагом, т.к. каждый слой состоит из n элеементов.
Звездочки в каждом слое начинаются с элемента, удовлетворяющего условию j >= n - i. Будем использовать это условие в if. В случае его выполнения выводим '*', иначе ' '.
после вывхода из внутреннего цикла переносим строку.

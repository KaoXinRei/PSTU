Дано число N, необзодимо напечатьать квадрат, длина стороны которого равна N.
Объявим переменную n и и запишем ввод пользователя в нее.
Чтобы пройтись по каждой строке квадрата будем использовать цикл for с i = 1, c ограничением i <= n, и шагом 1.
Внутри этого цикла, чтобы пройтись по всем элементам квадрата будем использовать цикл for с j = 1, ограничением j <= n и шагом 1
внутри этого цикла будем выводить '*', если j = 1, i = 1, j = n или i = n
После выхода из внутреннего цикла перенесем строку

Дано натуральное нечетное число N, большее 2, надо напечатать равнобедренныйы треугольник из звездочек с вершиной направленной вверх и основанием N.
Объявим переменную n и и запишем ввод пользователя в нее.
Далее нам понадобится не само чило N, а его цела часть от деления на 2, так что поменяем n на N / 2.
Данный треугольник будет иметь N / 2 + 1 слоев, поэтому будем использовать цикл for с i = 0, с ограничением i <= n и шагом 1.
Каждый слой треугольника заканчивается на позиции n + i + 1, поэтому будем использовать цикл for c j = 1, ограничением j <= n + i + 1 и шагом 1.
Каждый слой треугольника начинается с элемента, который удовлетворяет условию i + j > n. Таким образ внитри цикла будем проверять это условие, в случае его выполнения юудем выводить '*', иначе ' '
После вывхода из внутреннего цикла будем переходить на новую строку

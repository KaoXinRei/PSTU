В условии задачи дан закон, по которому считается S. Поскольку число слагаемых в законе не ограничено, поставим для программы ограничение n, значение которого попросим ввести пользователя.
Поскольку сумма состоит из n элементов, будем использовать цикл for c перемнной i = 1, ограничением i<=n и шагом 1
Каждый элемент суммы является произведением натуральных чисел от i до i*2 => для нахождения очередного элемента будем использовать цикл for с перменной j = i, условием j <=i*2 и шагом 1
Полученное число выведем на экран пользователя
